# CODELLAMA

This project is a simple AI chat app made using Python and Gradio. It uses Code Llama running on my computer through Ollama. When a user types a message, the app sends it to the local Ollama server and shows the model’s reply on the screen. Since the model is running on my local system , this project cannot be deployed online unless the model is hosted on a cloud server or replaced with an online API.
